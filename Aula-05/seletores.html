Seletores e Combinações CSS

Compreenda como selecionar elementos HTML para estilização

Tipos de Seletores

Seletores definem quais elementos HTML o CSS vai estilizar.

Seletor Universal

Aplica estilos a todos os elementos da página.

* {
  box-sizing: border-box;
  margin: 0;
  padding: 0;
}

Seletor de Tipo

Seleciona elementos pelo nome da tag HTML.

p {
  color: blue;
  margin-bottom: 10px;
}

Seletor de Classe

Seleciona elementos com um atributo class específico.

.destaque {
  background-color: #e6f7ff;
  border-left: 4px solid #1890ff;
  padding: 10px;
}

Seletor de ID

Seleciona um elemento com um ID específico (deve ser único na página).

#principal {
  font-weight: bold;
  color: #1890ff;
  font-size: 1.2rem;
}

Seletor de Atributo

Seleciona elementos com um atributo específico.

[type="text"] {
  border: 2px solid #52c41a;
  padding: 8px;
  border-radius: 4px;
  width: 100%;
}

Combinações de Seletores

Combinamos seletores para criar regras mais específicas.

Combinação Descendente

Seleciona elementos que estão dentro de outro elemento, em qualquer nível.

div p {
  color: red;
}

Combinação Filho Direto

Seleciona apenas os elementos que são filhos diretos de outro elemento.

div > p {
  color: blue;
}

Combinação Irmão Adjacente

Seleciona o elemento que vem imediatamente após outro elemento.

h1 + p {
  font-style: italic;
}

Combinação Irmão Geral

Seleciona todos os elementos que são irmãos de outro elemento e vêm depois dele.

h1 ~ p {
  padding-left: 15px;
}

Combinação Múltipla

Aplica o mesmo estilo a vários seletores diferentes.

h1, h2, h3 {
  font-family: 'Arial', sans-serif;
}

Resumo dos Seletores

  • * - Seletor Universal
  • p - Seletor de Tipo (tag)
  • .classe - Seletor de Classe
  • #id - Seletor de ID
  • [atributo] - Seletor de Atributo
  • seletor1 seletor2 - Descendente
  • seletor1 > seletor2 - Filho Direto
  • seletor1 + seletor2 - Irmão Adjacente
  • seletor1 ~ seletor2 - Irmão Geral
  • seletor1, seletor2 - Múltiplos Seletores
